# Course Seer App 说明

## 工作原理 How Does it Work：

Course Seer App 帮你实时关注你想选的课程的状态，一旦有已注册该课的人退出该课, Seer 便会通过邮件通知你。若是白金会员，Seer便会自动注册上该课（需提供Enterprise ID和密码），并且会通过电话通知你。
 
*注：必须要满足选课要求，Course Checker才能帮你选课。（例如只对部分专业的课）
